Assignment 6: Introduction to Metagenomics

Author

Arturo Torres Ortiz

Published

April 9, 2026

For this assignment, we will perform a microbiome composition analysis using shotgun sequencing metagenomics. We will use data generated from human fecal samples to compare the gut microbiota profiles of a healthy adult at baseline (Day 0) against their microbial profile immediately following a 4-day intervention with a cocktail of three last-resort antibiotics (Day 4). In total, we will have samples for days 0, 4, 8, 42, and 180.

The files used for the assignment follow the sample name structure: subject_timepoint_readpair.fq.gz. For instance, ERAS6_Dag4opt_wHAXPI002161-78.clean.pair.rmhost.1.fq.gz is subject 6, timepoint 4, forward read.

The original reference paper for this dataset:

  • Palleja, A., Mikkelsen, K.H., Forslund, S.K. et al. Recovery of gut microbiota of healthy adults following antibiotic exposure. Nat Microbiol 3, 1255–1265 (2018). https://doi.org/10.1038/s41564-018-0257-9

The authors utilized a deep shotgun metagenomic sequencing strategy to observe the effects of antibiotics in the gut microbiota in healthy individuals, allowing for high-resolution taxonomic profiling and the tracking of antibiotic resistance genes (the resistome) over a 6-month period.

We will process the raw metagenomic reads to identify differentially abundant microbial taxa and detect shifts in the resistome linked to antibiotic exposure.

Removing human reads

To remove human reads, we simply map reads to the human reference genome and take extract unmapped reads.

bwa mem 
<reference_genome>
<input_1.fq.gz> <input_2.fq.gz> | 
samtools view
-@ <n_threads>
-f12 -F256 - |
samtools sort
-@ <n_threads> -n -o unmapped.bam

samtools fastq -1 paired1.fq -2 paired2.fq -0 /dev/null -s /dev/null -n unmapped.bam

Removing low complexity reads

Highly conserved regions can cause missassignments during taxonomic classification. To solve this, we can remove regions in the genome with low entropy. We will use BBduck for this.

bbduk.sh
int=f
in=stdin.fq
out=stdout.fq
entropy=0.6
entropywindow=50
entropyk=5 |
gzip > output.fq.gz

Read level taxonomic assignment using Kraken2

A standard approach for taxonomic annotation of metagenomic sequences is using a k-mer based classifier. We will use Kraken 2, which breaks down our reads into short sequences (k-mers) and compares them against a massive database to assign taxonomy.

Because Kraken 2 assigns reads to various levels of the taxonomic tree (from Kingdom down to Strain), we will then use a companion tool called Bracken (Bayesian Reestimation of Abundance with KrakEN). Bracken takes the Kraken 2 output and pushes all the read counts down to a specific taxonomic level (e.g., the species level) to give us an accurate count matrix.

First, run Kraken 2 directly on your quality-controlled reads:

conda activate /project2/msalomon_1816/trgn_515/conda_env/kraken

kraken_db=/project2/msalomon_1816/trgn_515/6_metagenomics/k2_db

kraken2
  --db $kraken_db
  --threads $SLURM_CPUS_PER_TASK # Multithreading
  --paired <fwd.fastq.gz> <rev.fastq.gz>
  --report <sample_name>.kreport.tsv
  --output <sample_name>.kraken.tsv

Alpha diversity

Alpha diversity measures diversity within a sample.

Richness measures the number of distinct species within a sample. We can do this within R:

Within the vegan package you can also use:

sp_rich = specnumber(t(otu_table))

Analyze differences in alpha diversity

The Mann-Whitney U Test, also called Wilcoxon Rank Sum Test, is a non-parametric statistical test that is used to compare two groups, as we have here in our dataset. Being a non-parametric test, it does not assume normalcy in our data (unlike Student’s test). The test assesses if the two groups are from the same population, i.e. whether they are equal.

We will do it for the species richness.

sp_rich = apply(otu_table, 2, function(x) length(x[x != 0]))
df = data.frame('richness' = as.numeric(sp_rich), 'time' = trimws(gsub(".*_","",names(sp_rich))))
df$time = factor(df$time, levels = c(0, 4, 8, 42, 180))

We can only compare two groups using the Mann-Whitney U Test:

m1 = wilcox.test(as.numeric(richness) ~ time, data = df[df$time %in% c(0, 4),])

To compare more than two groups,we are going to employ the Kruskal-Wallis test, which is a further extension of the Mann-Whiteny U test for three or more groups. Again, we want to find out if the sample groups are representative of different populations, and therefore different from each other.

m2 = kruskal.test(as.numeric(richness) ~ time, data = df)

Using the Kruskal Test, we can only go so far as to say that there are differences between the groups. However, we don’t know exactly which groups are different. And, because we are comparing more than 2 groups, we must incorporate multiple test correction as well.Because the Kruskal-Wallis test uses a rank-sum approach, it is appropriate to use a post-hoc test which preserves these ranks, such as Dunn’s test (but an alternative could be the Conover-Iman test).

Instead of evaluating whether any of the groups are different, Dunn’s test uses pairwise comparisons to evaluate which groups might be significantly different from each other. And, we will use the Benjamini-Hochberg multiple test correction method to mitigate the false-discovery rate that is inherent to a series of pairwise tests.

m3 = rstatix::dunn_test(data = df, richness ~ time, p.adjust.method = "BH")

Beta diversity

Beta diversity measures diversity between samples.

First, let’s transform the abundances into relative abundances. This is not neccesarry with the functions we will use, but other functions may require it:

otu_rel = apply(otu_table, 2, function(x) x/sum(x))

Then we will calculate the Bray-Curtis dissimilarity:

bray_dist = as.matrix(vegdist(t(otu_rel), method = "bray"))
times =  sapply(colnames(bray_dist), function(x) strsplit(x, '_')[[1]][2])
times = factor(times, levels = sort(unique(times)))

Now that we have the raw distance data, we can do a permutational ANOVA (PERMANOVA) to see if there are significant differences. The PERMANOVA is used to analyze multivariate data that uses a dissimilarity metric. It is a consensus-based approach that uses the whole dataset and random subsets of the data to calculate a test statistic. In R, this can be done with the adonis2() function from the vegan package. PERMANOVA and Adonis tend to be used interchangeably to talk about the same test. To run our Adonis, use the following code:

adonis=adonis2(as.dist(bray_dist) ~ times)

Next, we will create an ordination and plot it to first visualize the differences between our groups.

library("ade4")
library("vegan")

meta = metaMDS(bray_dist, k=2) # k=2 to keep only 2 ordinations

s.class(
  meta$points, col = RColorBrewer::brewer.pal(length(levels(times)), 'Set1'), cpoint = 2, fac = times)

We can again conduct our PERMANOVA to compare all the groups in a pairwise manner.

devtools::install_github("pmartinezarbizu/pairwiseAdonis/pairwiseAdonis")
library('pairwiseAdonis')

pairwise=pairwise.adonis(as.dist(bray_dist), times, p.adjust.m = 'BH')
